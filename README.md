FoodApp
Descrição do Projeto
O FoodApp é um aplicativo de entrega de comida que conecta usuários a restaurantes locais, permitindo que eles explorem menus, façam pedidos e recebam suas refeições favoritas na porta de casa. Este projeto tem como objetivo proporcionar uma experiência conveniente e eficiente para os amantes da culinária.

Funcionalidades Principais
Exploração de Restaurantes: Os usuários podem navegar por uma lista de restaurantes locais, filtrar por tipos de culinária e visualizar detalhes do menu.

Realização de Pedidos: Possibilidade de adicionar itens ao carrinho, personalizar pedidos e efetuar o pagamento de forma rápida e segura.

Acompanhamento de Pedidos: Os usuários podem rastrear o status de seus pedidos em tempo real, desde a preparação até a entrega.

Avaliações e Comentários: Capacidade de deixar avaliações e comentários sobre restaurantes e pratos, proporcionando uma comunidade interativa.

Tecnologias Utilizadas
Frontend: React Native
Backend: Node.js, Express
Banco de Dados: MongoDB
Autenticação: JWT (JSON Web Token)
Pagamentos: Integração com serviço de pagamento online (ex: Stripe)
